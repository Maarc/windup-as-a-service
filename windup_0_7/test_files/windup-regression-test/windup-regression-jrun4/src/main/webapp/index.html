JRun 4 samples server

The samples JRun server contains a variety of applications that illustrate a number of coding techniques. The samples applications included with JRun are explained in the following table:

Application Description
Compass Travel A simple online travel agent trip reservation system. This application illustrates JSP, servlet, and EJB programming and is delivered as an enterprise application in an open-directory structure.
TravelNet An online travel agent that sells trips from compass travel. This application illustrates JSP and web services programming and is delivered as an enterprise application in an open-directory structure.
WorldMusic An e-commerce web application that sells music. This application also contains an admin module. The world music application illustrates many common design patterns using JSP, servlet, and JavaBean programming. Delivered as a compressed WAR file.
JRun programming techniques Illustrates programming techniques described in the JRun Programmer's Guide.
Web services Illustrates programming techniques used in JRun web services programming.
Flash Remoting Contains Flash movies and adapters that illustrate how to use Flash Remoting with JRun. The Flash Remoting service is the server-side runtime software that processes requests made from Flash clients. Because Flash Remoting is a feature of JRun 4, the Flash Remoting service is always running wherever JRun 4 is installed.
Java Smart Ticket Java Blueprints Java 2 Platform, Micro Edition (J2ME) application.
JSP Standard Tag Library (JSTL) JSTL 1.0 samples and links to other JSTL resources.
Java Pet Store Java Blueprints J2EE application. For instructions on running Java Pet Store with JRun (and to see a running version of the J2EE Blueprints application) see http://blueprints.macromedia.com/,
